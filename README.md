# LotttttoY

**LotttttoY** — это Android-приложение для майнинга криптовалюты Monero (XMR) на мобильных устройствах. Оно поддерживает два режима работы: соло-майнинг и майнинг в пуле.

## Особенности:
- Совместимость с большинством Android-устройств.
- Простота настройки и удобная панель управления.
- Возможность выбора оптимального режима работы (соло или пул).
- Поддержка популярного майнера XMRig.

## Требования:
- Android 5.0 (API Level 21) и выше.
- Доступ к интернету для подключения к пулу.

## Установка:
1. Скачайте и установите APK-файл.
2. Откройте приложение и введите свой адрес кошелька.
3. Нажмите кнопку «Start Mining», чтобы начать майнинг.
